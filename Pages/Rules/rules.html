📕Правила аналитики Fireplace

Данный свод правил является рекомендацией, в случае конфликтных ситуаций мы будем обращаться именно сюда если кланы сами этого захотят. Правила еще пишутся! Некоторые пункты будут изменять или добавляться

Раздел: ⛊Кланы
0.1

Создать официальный клан можно сделав запрос в аналитике (указав название, тег, цель клана, никнейм лидера) или же на официальном сайте. После одобрения аналитикой, вы сможете создать канал в любой соц.сети, дав ссылку непосредственно на канал клана

0.2

Все кланы(которые согласны обнулить победы) начинают свои победы с нуля, подсчитывать победы будет администрация, это предотвратит фальшивые и липовые победы

0.3

Рейтинг кланов: Рейтинг клана(Престиж) - новая валюта, она есть только у кланов входящих в Топ5. Чем больше, тем больше привелегий. Если рейтинг будет отрицательным, клан и его участников могут понизить в топах и недопустить к некоторым турнирам

0.4

В клане может находиться максимум 10 (активных) игроков. Минимум - 1. Клан может иметь приписку, должности участников внутри клана и подгруппу

0.5

Заморозка клана осуществляется только Лидером или заместителем клана, возможна лишь на пару дней(максимум неделя) по уважительной причине

Раздел: Солокв🗡️
0.6

Соло кв играется по согласию 2 сторон, на любых оружиях, включая арендованные, если одна из сторон против, оружия выбираются по дефолту (AK-47, M4A1)

0.7

Любые выходы с лоу хп запрещены, если игрок все же вышел, другой команде присваевается +1 очко. Больше 3 раз, автоматическое поражение. Если появилось сообщение что игрок убился, другой команде даётся +1 очко.

0.8

После окончания любого раунда, команде которая была за определенную команду, нужно перейти на противоположную.

0.9

Если игрок одной из сторон вышел из сервера, запускается отсчёт в 3 минуты. По истечении времени присваивается автоматическая победа другой стороне. В случае уважительной причины, назначается переигровка раунда

1.0

Укрытие от противника путём забрекодирования себя, автоматически будет зачтено поражение стороне которая пряталась

1.1

солокв может быть только форматом 1×1, замена игрока невозможна в случае начала игры, разрешается только при договорённости 2-х сторон(замена после окончания 1-го раунда или до игры)

1.2

намеренный выход чтобы оставить счёт за 1-2мин до конца раунда, карается автопоражением. Переигровка не может быть осуществлена.

1.3

Игра с наблюдателями возможна при согласовании 2-х сторон (на администрацию это правило не работает)

Раздел: Дуо-трио+ кв⚔️
1.4

Если игрок одной из команд не зашёл на сервер, убийство не может быть совершено. В случае нарушения — другой команде +1 очко. При повторных нарушениях выдается АП

1.5

замена 1 игрока(2 заменить нельзя) невозможна в случае начала игры, разрешается только при договорённости 2-х сторон(замена игрока после окончания 1-го раунда или до игры)

1.6

Дуо(и более) кв играется по согласию 2 сторон, на любых оружиях, включая арендованные, если одна из сторон против, оружия выбираются по дефолту (AK-47, M4A1)

Раздел: КВ на топ (по кланам)
1.7

Запрещены к использованию арендованные оружия:

  • Class A — MP5-mod | M4A1-Mod | SCAR | KRISS Vector | RPK-74 | P90 | M249 SAW
  • Class B — AWP | DSR-1 | SVD | VSS
  • Class C — Mossberg | Saiga-12
  • Class D — Mac-10 | Colt Python | Mac-11
  • Class E — 40mm GL | RPG-7
  • Class G — Karambit | Katana
1.7.1

Разрешённые оружия:

  • Class A — MP5 | AK-47 | M4A1
  • Class B — M24
  • Class C — Shotgun
  • Class D — Beretta | Desert Eagle | Tec-9
  • Class E — Grenade
  • Class G — M9 Baylet | Military Shovel | Fire Axe
1.8

Чтобы войти в топ и занимать более высокую позицию, в клане должно быть минимум 2 игрока.Если 1 игрок, клану даётся 3 дня на нахождение игрока(в клан). Спустя 1 день участнику будет доступны игры на уничтожение и на топ

1.8.1

Чтобы попасть в топ-лист, необходимо кинуть топ 10 клану запрос на игру. Игра на топ по кланам проходит в режиме "лесенка" с позиции на которую вас поставили по очерёдно на верхнюю

1.8.2

Кидать кв на топ можно 3 раза, после 3-х отказов автоматическая победа клану запросившему игру, после 1-го отказа можно повторно кинуть через 2 дня (1день для кланов не входящих в топ 1-3)

1.9

Помощь, замена или новая личность игрока из другого клана(не подгруппы и альянса) карается автопоражением без права на отыгровку

2.0

Клан может взять у своей подгруппы или альясна участников на кв, но важно что хотя бы 1 игрок был из основного клана

Раздел: Топ (по игрокам)
не составлен
Раздел: КВ на уничтожение💀
2.1

Клан может взять у своей подгруппы или альясна участников для КВ на уничтожение, но важно что хотя бы 1 игрок был из основного клана

2.1.1

если участвуют несколько кланов(с одной или другой стороны) им необходимо быть в альянсе, в ином случае игру признают недействительной. Администрация должна быть уведомлена (по запросу) об составе обоих кланов

2.2

Если клан участвует в турнире | игра на уничтожение проходит без администрации аналитики | нету разрешения заместителя(если он имеется) или главы клана игра не может быть осуществлена

2.3

Запрещены к использованию арендованные оружия:

  • Class A — MP5-mod | M4A1-Mod | SCAR | KRISS Vector | RPK-74 | P90 | M249 SAW
  • Class B — AWP | DSR-1 | SVD | VSS
  • Class C — Mossberg | Saiga-12
  • Class D — Mac-10 | Colt Python | Mac-11
  • Class E — 40mm GL | RPG-7
  • Class G — Karambit | Katana
2.3.1

Разрешённые оружия:

  • Class A — MP5 | AK-47 | M4A1
  • Class B — M24
  • Class C — Shotgun
  • Class D — Beretta | Desert Eagle | Tec-9
  • Class E — Grenade
  • Class G — M9 Baylet | Military Shovel | Fire Axe
2.4

После поражения одной из сторон у клана аннулируются все победы и снижается престиж(можно восстановить при согласовании победившей стороны) и клан закрывается

2.4.1

Лидеру клана после игры на уничтожение, в котором они потерпели поражение запрещается создавать клан в течении 3 дней, после этого он может восстановить свой клан
